Plugin simples para WordPress que facilita a criação e gerenciamento de redirecionamentos de URLs de forma intuitiva e segura.

== Description ==
O plugin \"Redirect Simple\" para WordPress oferece uma solução eficiente e acessível para configurar e gerenciar redirecionamentos de URLs diretamente do painel administrativo. Com uma interface intuitiva, permite aos usuários criar múltiplos redirecionamentos personalizados com facilidade, garantindo uma navegação mais fluida para os visitantes do site. Além disso, incorpora medidas de segurança robustas para proteger as URLs redirecionadas e suporta a personalização avançada das configurações de redirecionamento, tornando-o ideal para qualquer site que necessite de um controle preciso sobre seu fluxo de tráfego.

== Installation ==
- Faça o download do arquivo zip do plugin \"Redirect Simple\".
- Faça login no painel administrativo do seu site WordPress.
- Vá para Plugins > Adicionar Novo.
- Clique em Enviar Plugin e selecione o arquivo zip que você baixou.
- Clique em Instalar Agora e depois em Ativar Plugin.
